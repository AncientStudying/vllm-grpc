|--------|------|

## State Transitions

No stateful entities exist in Phase 1. Both services are stateless request-handlers.

The only observable state is **service availability**:

```
Frontend UP  → Proxy /healthz → HTTP 200, {"status": "ok"}
Frontend DOWN → Proxy /healthz → HTTP 503, {"status": "error", "detail": "..."}
```
